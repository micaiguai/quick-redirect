| Platform        | Shortcut           | Action                    |
|-----------------|--------------------|---------------------------|
| Windows / Linux | `Ctrl + Shift + P` | Open Quick Redirect Popup |
| macOS           | `Cmd + Shift + P`  | Open Quick Redirect Popup |
